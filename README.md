## Quick start
#### Create from template using GitHub CLI
```terminal
gh repo create [<name>] -p Cuppachino/vite-react-ts-tailwind
```
#### Or clone locally
```terminal
gh repo clone Cuppachino/vite-react-ts-tailwind [<directory>]
```

#### Install dependencies & start dev server
```terminal
// YARN:
yarn
yarn dev
```
```terminal
// NPM:
npm i
npm run dev
```
